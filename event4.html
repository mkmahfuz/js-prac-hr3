This is an example of event bubbling

Bubbling:(UP) Inner first then Parent

Capturing or Trickle: (Down) Parent first then Inner

To stop propagation use: event.stopPropagation() or event.stopImmediatePropagation();

stopPropagation() : it will stop upper event

stopImmediatePropagation() : it will stop upper event, also stop anyother events of this element
