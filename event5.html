Notes: console.log(this)--> here this is the item itself

event.target --> here target is the clicked item

so for getting innerText we can use event.target.innerText

we can pass the function parameter in addEventListener, the parameter can be "event" or "anyting"

here all are same::: item.addEventListener('click',function(){console.log(this)}); item.addEventListener('click',function(event){console.log(event.target)}); item.addEventListener('click',function(any){console.log(any.target)});
